# 多元对话 — 多角色AI心理陪伴系统 🧠

> 🚧 **公测中** · 你的每一条反馈都在帮它变得更好

一个可以自由切换六位AI专家进行深度对话的网页应用。四位不同流派的心理学家 + 历史学家 + 经济周期分析师，每人都有完整的人设背景、专业临床框架和对中国社会的深度理解。

## ✨ 功能

- **6位AI角色** — 精神分析学家深源、CBT治疗师明思、人本咨询师心然、存在治疗师存光、历史学家观澜、周期分析师知行
- **双模式切换** — 咨询模式进行深度对话，分析模式获得专业解读
- **Zootopia风格动物头像** — 考拉/赤狐/小熊猫/大象/猫头鹰/鹿
- **真实人物设定** — 每位角色有完整的人生经历、专业受训背景、治疗框架和核心技术
- **中国社会语境** — 理解高考压力、内卷、996、代际期待、面子文化等本土议题
- **打字机回复效果** — AI消息逐字显示，自然对话体验
- **记忆与学习系统** — 对话自动保存、回复点赞/点踩、高评分示例自动优化、数据看板
- **一键部署** — 基于 DeepSeek API，支持 Zeabur 云部署

## 🚀 快速部署

```bash
pip install -r requirements.txt
export DEEPSEEK_API_KEY="你的密钥"
python server.py
```

或上传到 Zeabur 一键部署（需要 `Procfile` + `requirements.txt` + `server.py` + `multi-role-ai-chat-v2.html`）。
